# 创建应用

---

## Create React App

`Create React App` 是一个用于学习 `React` 的舒适环境，也是用 `React` 创建新的单页应用的最佳方式。

---

你需要在你的机器上安装 `Node >= 14.0.0` 和 `npm >= 5.6`。要创建项目，请执行：

```shell
npx create-react-app my-app
cd my-app
npm start
```
